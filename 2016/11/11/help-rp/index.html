记住密码帮助文档

发表于 2016-11-11

Absolute security does not exist. Security is a set of measures, being piled up and combined, trying to slow down the inevitable.

没有绝对的安全。所谓的安全是指利用一系列措施的堆积和组合，来试图延缓必然发生的事情。

关于安全

该软件不保证绝对安全，请勿存储金钱等重要相关信息。

该软件只有一个重要权限：读写文件，其他没有任何与外界操作的权限（联网、发短信等）。该网页内容是从外部浏览器打开，而非app具有网络权限。

该软件使用的数据库非Android系统自带的Sqlite数据库，而是使用流行的Realm数据库，更加快速、安全。且数据库经过标准的AES256加密。所以每次升级程序前请务必请手动备份文件！防止不兼容等问题导致数据遗失！

功能

  • 自毁模式谨慎开启，防止朋友打开你的app乱输入密码导致数据被清空，而你却没有备份！
  • 明文备份所有数据，可选择任意文件夹存储，单个条目的自定义字段备份的时候是写在note里面的。

对于自动备份，该版本（v2.0.0）已关闭，下一版本会改进。

  • 由于软件没有联网功能，我的测试手机有限，所以如果有bug希望将具体操作步骤反馈给我排查改进。
  • 主页的底部菜单栏在主页向下滑动会自动隐藏，向上滑动一点就会出现
