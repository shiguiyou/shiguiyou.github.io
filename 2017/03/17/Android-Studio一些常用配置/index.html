我的Android Studio一些配置

发表于 2017-03-17

本来一直用的自己的mac pro开发，公司突然给配了一台Windows，想着能用就用吧，翻墙，下载AS，码代码，郁闷。。。8g内存又卡又慢，还是新电脑！不过google配置快捷键的时候发现了一篇很好的文章，可以让你的开发工作事倍功半（机智），下面就摘取一些我觉得蛮好用的分享出来。

Log颜色显示

Preferences (Settings on Windows / Linux machines) → Editor → Colors & Fonts → Android Logcat

  • Assert #BA68C8
  • Debug #2196F3
  • Error #F44336
  • Info #4CAF50
  • Verbose #BBBBBB
  • Warning #FF9800

使用模版

快捷键：cmd + j (Windows / Linux: ctrl + j)

打开一个class文件

快捷键：cmd + o (windows / Linux: ctrl + n ).

非常好用，不用打开文件目录一层一层去找那个文件

选择最后几条复制内容粘贴

快捷键：cmd + shift + v (Windows / Linux: ctrl + shift + v)

默认的有5条

关闭当前tab

快捷键：cmd + w (Windows / Linux: ctrl + F4)

格式化代码

快捷键：cmd + option + l (Windows / Linux:
ctrl + alt + l)

编译并运行

快捷键：control + r (Windows / Linux: shift + F10)

常用插件

  • Android ButterKnife Zelezny
  • .ignore
  • GsonFormat
  • Key promoter
  • MarkDown
